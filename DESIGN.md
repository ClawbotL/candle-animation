# Candle Animation Upgrade - Design.md

## Requirements Addressed
1. **Enhance wax texture and translucency**
2. **Improve flame physics and color gradients**
3. **Refine melt pool and wax drip details**
4. **Improve ambient lighting and reflections**

## Design Overview

### 1. Enhanced Wax Texture and Translucency
- Add SVG feTurbulence noise for a realistic wax grain texture
- Improve the translucency gradient at the top of the candle, making it stronger where the flame glows
- Add subtle variations in wax color and reflection to mimic real wax imperfections

### 2. Improved Flame Physics and Color Gradients
- Enhance the flame flicker algorithm with more natural, organic movements
- Add dynamic flame response that slightly changes shape and intensity over time
- Improve color gradients for more depth, with smoother transitions between layers
- Add slight color temperature shifts to the flame layers for added realism

### 3. Refined Melt Pool and Wax Drip Details
- Improve the melt pool shape with more natural curvature and subtle reflections
- Add additional wax drips with more realistic physics (varying sizes, shapes, and speeds)
- Enhance the wick interaction with the melt pool, adding slight surface ripples

### 4. Improved Ambient Lighting and Reflections
- Add subtle dynamic reflections of the flame on the wax body
- Enhance the ambient glow with more realistic falloff
- Improve the surface interaction, adding a slight warm reflection of the candle on the table
